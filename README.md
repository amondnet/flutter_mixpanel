# flutter_mixpanel

Flutter Mixpanel Plugin using Mixpanel SDK

## android

- mixpanel-android:5.+

## iOS

- mixpanel sdk: [mixpanel-swift:2.8.1](https://github.com/mixpanel/mixpanel-swift/releases/tag/v2.8.1)
- swift : 5+
- deployment target : 9.0  
- support bool type 

## Supported Data Type

- String
- Numeric
- Boolean
- Date
- List
- Uri
