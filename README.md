# flutter_mixpanel

Flutter Mixpanel Plugin using Mixpanel SDK

## android

- mixpanel-android:5.+

## iOS

- mixpanel sdk: [mixpanel-swift:2.7.2](https://github.com/mixpanel/mixpanel-swift/releases/tag/v2.7.2)
- swift 5
- support bool type 

## Supported Data Type

- String
- Numeric
- Boolean
- Date
- List
- Uri
